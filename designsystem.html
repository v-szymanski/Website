REVVITY INTERNSHIP

Building a Custom Design System for Medical Applications

Overview

This project is an ongoing effort at Revvity to create one source of truth for all product design in the company, encompassing software, hardware, and reagents. Efforts have been underway for multiple years, but the beginning of a scalable and cohesive design system with documentation began in 2022, during my first internship at the company. The project has since evolved into a company-wide effort involving the entire design team, senior leaders, developers, and product managers. The public documentation website for this design system can be found at revvity.design.
